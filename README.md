# PixiJS Space Dodge

Small PixiJS arcade game prepared for GitHub Pages deployment.

## Run Locally

Open `index.html` in a browser, or serve the folder with a simple static server.

## Deploy To GitHub Pages

1. Create a new GitHub repository.
2. Upload this project to the repository root.
3. Make sure your default branch is `main`.
4. Open `Settings -> Pages` in GitHub.
5. Under `Build and deployment`, choose `GitHub Actions`.
6. Push to `main` and wait for the `Deploy to GitHub Pages` workflow to finish.
7. Your game will be available at `https://<your-username>.github.io/<repo-name>/`.

## Notes

- The project is deployed as a static site, so no build step is required.
- Keep `index.html`, `js/`, and `assets/` in the repository root so relative paths keep working.
